Developer Information

To use the REST API, one needs an account with eInfraCentral.
A subsequent POST request to beta.einfracentral.eu:8080/login with a JSON body containing fields email and password issues a jwt cookie.
Any subsequent request to the API using that cookie, is authenticated.
The complete eInfraCentral API documentation is a living document, generated using frameworks compliant with OpenAPI Specification 2.0
